 🏋️ Fitness Tracker Backend

A **Spring Boot REST API** for a Fitness Tracker application that allows users to register, authenticate, manage workouts, track fitness progress, and securely access resources using JWT Authentication.

---

## 🚀 Features

- 🔐 User Authentication & Authorization (JWT)
- 👤 User Registration & Login
- 🏋️ Workout Management (CRUD)
- 📊 Track Fitness Activities
- 🔒 Role-Based Access Control (Admin/User)
- 📄 RESTful APIs
- 📖 Swagger API Documentation
- 🗄️ MySQL Database Integration
- ☁️ Docker Support
- 🌐 Deployed on Render

---

## 🛠️ Tech Stack

### Backend
